# magazay 0.3
Un projet pour la gestion des stocks d'épicerie dans une application web.

# Déscription
Ce projet permet de gérer et simuler les éléments retrouvés dans les simples épiceries dans une application web par l'administrateur.
Liste, modifie, efface, rajoutte dans les éléments des tables fournisseurs, stock et produits.
Il permet la suivi des produits en rupture de stock et en chemin de rupture.

## Installation locale
1. Clonez le repository : 
    (exemple) git install https://github.com/nom-utilisateur/nom-projet.git
2. Exportez la base de donnée dans votre server web :
    WAMP ou XAMP, faite reférence à la documentation
3. Placer le fichier télécharger dans le repository de votre server :
    XAMP : .../htdocs/mon-projet
    WAMP : .../www/mon-projet

## Utilisation
1. Lancer le serveur web
2. Ouvrez le navigateur et allez à http://localhost/nom-projet/
3. Explorez
